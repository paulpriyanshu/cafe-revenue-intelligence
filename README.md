## 🎯 Project Overview

This project focuses on transforming "dirty" cafe sales data into a strategic asset. By analyzing transaction patterns, product performance, and temporal trends, it provides cafe owners with data-driven insights to maximize revenue and streamline staffing.

## 🚀 Key Features

### 1. Advanced Multi-Horizon Forecasting
Powered by an **XGBoost Regressor**, the system provides high-accuracy sales predictions across three critical horizons:
*   **Daily**: Prep for tomorrow's inventory.
*   **Weekly**: Plan staff schedules.
*   **Monthly**: Analyze seasonal growth trends.

### 2. Strategic Product Analysis (Pareto 80/20)
Identifies your most valuable items through automated tiering:
*   **A-Tier (Core Bestsellers)**: The 20% of items driving 80% of revenue.
*   **B-Tier (Growth Opportunities)**: High-potential items for bundling.
*   **C-Tier (Underperformers)**: Candidates for menu rotation.

### 3. Operational Intelligence
Translates complex heatmaps into simple business tactics:
*   **Peak Period Staffing**: Identifying exactly when to "double-staff" to capture every sale.
*   **Gap Marketing**: Recommendations for "Happy Hour" or student discounts during identified slow hours.

### 4. Robust Data Cleaning Pipeline
Handles common real-world data issues:
*   Standardizing `Transaction Date` formats.
*   Fixing pricing errors and missing value imputations.
*   Location-based sales segmentation (In-store vs. Takeaway).

## 🛠️ Tech Stack

- **Languages**: Python
- **Data Handling**: Pandas, NumPy
- **Visualization**: Seaborn, Matplotlib
- **Machine Learning**: XGBoost, Scikit-learn

## 📈 Impact
By implementing the forecasts and operational measures in this project, a cafe can:
- **Reduce Food Waste**: Align prep with predicted demand.
- **Improve Customer Experience**: Optimize staffing for busy windows.
- **Increase AOV (Average Order Value)**: Use Pareto insights for strategic cross-selling.
